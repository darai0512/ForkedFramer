## レイヤー: 選択操作の動作

| 操作 | 動作 |
|---|---|
| **クリック** | 単一選択（他の選択は解除） |
| **Ctrl/⌘ + クリック** | トグル（個別に追加/削除） |
| **Shift + クリック** | 前回選択したレイヤーから現在のレイヤーまで範囲一括選択 |
| **Shift + Ctrl/⌘ + クリック** | 既存の選択を維持したまま範囲追加 |
| **「全選択」ボタン** | 全レイヤーを選択（レイヤー2つ以上で表示） |
| **「全解除」ボタン** | 全選択状態のとき表示、全選択を解除 |

## File System Access API (showOpenFilePicker)
- ✅ Chrome/Edge（デスクトップ）→ 利用可能
- ❌ Firefox / Safari / HTTPSではないサイト → 利用不可
- ❌ HTTP環境（セキュアコンテキスト外）→ 利用不可

## 以前の素材集機能: 画像登録後、
- 別のコマ・ページにドラッグ＆ドロップ 素材集パネル上に並んでいる使いたい画像を、そのままキャンバス上の 入れたいコマに向かってドラッグ＆ドロップ してください。これでローカルから読み込む手間なく直接コマに適用されます。
- ふきだし/オブジェクトとしてポンと置く 素材集パネル上の画像を（ドラッグではなく）ただクリックするだけで、今見ているページの空いている場所に「画像オブジェクト（吹き出し扱い）」として即座に配置されます。背景がないキャラクター画像などをポンポン置きたいときに便利です。 

## 各AIツールの解説

すべての AI 系ツールは **Supabase Edge Functions** をバックエンドとして使用する非同期処理でした。フロントエンドは「**beforeRequest** → APIリクエスト送信 → **afterRequest** → ポーリングで結果取得」というパターンで動作します。

### 各ツールの動作

| ツール | アイコン名 | 動作概要 | ログイン必要 |
|--------|-----------|----------|------------|
| **消しゴム** (eraser) | eraser | ブラシで塗った領域を **AIインペインティングで消去**。`imageMask` ダイアログでマスク領域を手書き指定 → Supabase `eraser` Edge Function へ送信 | ✅ |
| **動画作成** (video) | video | 静止画から **AI Image-to-Video** を生成。`videoGenerator` ダイアログでパラメータ指定 → `image2Video` Edge Function → 数分かかるため非同期ポーリング | ✅ |
| **背景除去** (punch) | punchIcon | 画像の背景を **AIで自動除去**（removebg）。ダイアログなし、即時リクエスト送信 | ✅ |
| **アップスケール** (upscale) | upscale | 画像解像度を **AI超解像**で拡大 | ✅ |
| **インペイント** (inpaint) | inpaint | マスク領域指定で **AI塗りつぶし生成** | ✅ |
| **アウトペイント** (outpaint) | outpainting | コマからはみ出た余白を **AI生成で補完**。コストが0だと使用不可（コマに余白がない場合） | ✅ |
| **テキスト消しゴム** (textedit) | textedit | 吹き出しのテキスト部分を認識して消去（`texteraser`） | ✅ |
| **角度補正** (angleedit) | angleedit | 画像の傾きを補正 |  ✅ |
| **テキスト分離** (textlift) | textlift | 画像中のテキストを認識し、テキスト部分を除去したクリーン版レイヤーを追加 | ✅  |
| **レイヤー化** (layerize) | layerize | 画像内の人物・背景などを個別レイヤーに分解（複数Film生成） | ✅  |
|「吹き出し化」(textlift)| textLiftFilm.ts |画像内のテキストをOCRで検出→吹き出し(Bubble)として自動配置する機能| ✅ 
| **複製** (duplicate) | duplicate | フィルムをそのままコピーしてスタックに追加（ローカル処理のみ） | ❌ |
| **素材集へ** (sendToMaterial) | stamp | 現在の画像を素材集フォルダへ保存 | ❌ |
| **ダウンロード** | download | 現在のレイヤー画像をPNG/MP4でローカル保存 | ❌ |

吹き出し化:
- ユーザーが検出結果を選択・編集（テキスト内容、縦書き/横書き）
- 選択された各テキスト領域ごとに Bubble オブジェクトを自動生成し、元の画像上の位置に正確に配置
- オプションで元画像からテキストを消去（Supabase textEraser API で消去済画像を合成）
