新機能についてのツイート用テキストを作成してください: $ARGUMENTS

引数に書かれた機能の説明を元に、以下のフォーマットでツイート用テキストを生成してください。

## フォーマットルール
- 各機能を簡潔に1〜2行で説明（絵文字不要）
- 機能が複数ある場合は空行で区切る
- 末尾に以下のハッシュタグを付ける：

```
#FramePlanner
#まんがファーム
```

- 全体で280文字以内に収める（日本語は1文字=2文字換算）
- 技術的な用語は避け、ユーザー視点で書く
- 敬語（です・ます調）で統一

## 出力例

```
レイヤー素材の個別ダウンロード時、ファイル名がp001_f002_l001.pngのような形式になりました

ダウンロードメニューに「使用素材の一括ダウンロード」を追加しました

#FramePlanner
#まんがファーム
```

## 出力方法
生成したテキストをそのまま表示してください。ユーザーが /copy でコピーできます。
